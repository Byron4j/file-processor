## 📖 演示流程指南

### 第一步：访问系统

打开浏览器访问：**http://localhost:3000**

> 注意：首次访问会跳转到登录页面

---

### 第二步：登录系统

由于后端使用内存数据库(H2)，需要注册用户或使用默认测试账户。

#### 方式1：通过API注册用户

```bash
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "admin",
    "password": "123456",
    "email": "admin@example.com"
  }'
```

#### 方式2：使用后端测试接口

后端提供了测试用的初始化数据，启动时会自动创建测试用户。

**测试账户：**
- 用户名: `admin`
- 密码: `admin123`

---

### 第三步：文件管理演示

#### 3.1 上传文件

1. 进入「全部文件」页面
2. 点击「上传」按钮或拖拽文件到页面
3. 支持多种文件格式：
   - 文档：PDF, DOC, DOCX, TXT
   - 图片：JPG, PNG, GIF, WebP
   - 视频：MP4, WebM, MOV
   - 音频：MP3, WAV, OGG
   - 压缩包：ZIP, RAR, 7Z

#### 3.2 文件预览

点击文件名称进入预览页面：

| 文件类型 | 预览功能 |
|---------|---------|
| PDF | 分页浏览、缩放、旋转、下载 |
| 图片 | 缩放、旋转、全屏查看 |
| 视频 | 在线播放、画质切换、倍速播放 |
| 音频 | 播放控制、进度条 |
| Office | DOCX/XLSX/PPTX 转 HTML 预览 |

#### 3.3 文件操作

- **重命名**：右键菜单或点击更多按钮
- **移动**：选择目标文件夹
- **复制**：创建文件副本
- **删除**：移动到回收站
- **收藏**：添加到收藏夹
- **分享**：生成分享链接

---

### 第四步：格式转换演示

#### 4.1 文档转换

路径：`/convert/document`

支持转换：
- DOC ↔ DOCX
- DOCX → PDF
- DOCX → TXT
- DOCX → HTML

#### 4.2 图片转换

路径：`/convert/image`

支持格式：
- JPEG, PNG, GIF, BMP, WebP, TIFF

可选参数：
- 质量调整
- 尺寸缩放
- 裁剪模式

#### 4.3 视频转换

路径：`/convert/video`

支持格式：
- MP4, WebM, MOV, AVI, MKV

可选参数：
- 分辨率 (1080p/720p/480p)
- 视频编码
- 比特率

---

### 第五步：PDF 工具演示

路径：`/tools/pdf`

#### 功能列表：

1. **PDF 合并**
   - 上传多个 PDF 文件
   - 调整合并顺序
   - 生成合并后的文件

2. **PDF 拆分**
   - 按页码范围拆分
   - 每 N 页拆分

3. **PDF 旋转**
   - 90°/180°/270° 旋转
   - 指定页面范围

4. **PDF 水印**
   - 文字水印（支持透明度、位置）
   - 图片水印

5. **PDF 加密**
   - 设置用户密码
   - 设置权限（打印、复制、编辑）

---

### 第六步：付费订阅演示

路径：`/subscription`

#### 6.1 查看套餐

1. 进入「订阅套餐」页面
2. 查看可用的订阅计划：
   - 免费版：基础功能，有限额度
   - 基础版：¥29/月，适合个人用户
   - 专业版：¥99/月，适合专业用户
   - 企业版：¥299/月，适合团队使用

#### 6.2 选择套餐

1. 选择计费周期（月付/年付）
2. 点击「立即订阅」

#### 6.3 支付

支持两种支付方式：
- **支付宝**：跳转支付宝收银台完成支付
- **微信支付**：扫码完成支付

支付成功后自动激活订阅。

#### 6.4 秒传功能

系统支持文件秒传功能：
1. 上传文件时自动计算MD5指纹
2. 如果文件已存在于系统中，直接秒传完成
3. 节省上传时间和流量

---

### 第七步：AI 功能演示

路径：`/ai/*`

#### 7.1 智能摘要

1. 上传文档（PDF/DOCX/TXT）
2. 选择摘要风格（简洁/详细/要点）
3. 设置最大字数
4. 点击生成摘要

#### 7.2 文档问答

1. 上传文档
2. 在对话框中输入问题
3. AI 基于文档内容回答

示例问题：
- "这份文档的主要内容是什么？"
- "文档中提到了哪些关键数据？"
- "总结一下第三章的内容"

#### 7.3 文档分类

系统自动识别文档类型：
- 合同 (CONTRACT)
- 发票 (INVOICE)
- 报告 (REPORT)
- 简历 (RESUME)
- 其他

#### 7.4 智能标签

基于文档内容自动生成关键词标签

## 🐛 常见问题

### Q1: 无法登录
**解决**: 检查后端是否正常运行，尝试调用注册API创建用户

### Q2: 文件上传失败
**解决**: 检查文件大小是否超过100MB限制

### Q3: 预览不工作
**解决**: 确认文件格式是否受支持，检查浏览器控制台错误信息

### Q4: AI 功能不可用
**解决**: AI 功能需要配置 API Key，请在 `application.yml` 中配置
